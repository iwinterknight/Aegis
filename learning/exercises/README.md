# `learning/exercises/` — Programming Exercises

Coursera-style, hands-on exercises subdivided by build phase. Each file (notebook or script) fully
explains the surrounding code, then has you implement the marked gaps:

```python
# Spec: return SUM(amount) over rows where account_type == target, as a Decimal.
# ==== YOUR CODE HERE (start) ====
...                      # you write this
# ==== YOUR CODE HERE (end) ====
```

Every exercise is runnable end-to-end and verifies your work with `assert`-based self-checks before
you consult `../solutions/`. Reference solutions are deliberately kept out of these files.

Subfolders: `phase-<N>-<slug>/`. See [`../README.md`](../README.md) for full conventions.
