# `learning/quizzes/` — Checkpoint Quizzes

Short, targeted quizzes (Markdown) that lock in the foundations before we move from learning to
building. Subdivided by build phase: `phase-<N>-<slug>/`.

**Format (standard):** **objective multiple-choice** — each question marked **(select one)** or
**(select all that apply)** — with answers collapsed under a `<details>` block. No free-text writing;
you pick letters and verify. Quizzes target understanding of *mechanism, tradeoffs, and principles*,
not memorization or terminology (in particular, never domain jargon — e.g. no finance-term recall).
Each answer names the transferable principle underneath.
