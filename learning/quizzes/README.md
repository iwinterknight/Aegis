# `learning/quizzes/` — Checkpoint Quizzes

Short, targeted quizzes (Markdown) that lock in the foundations before we move from learning to
building. Subdivided by build phase: `phase-<N>-<slug>/`.

Format: a handful of conceptual + applied questions, answers collapsed under a `<details>` block so
you can self-test first. Quizzes target understanding of *mechanism and tradeoffs*, not memorization.
