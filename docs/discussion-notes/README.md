# `docs/discussion-notes/` — Deep-Dive Discussion Logs

A running log per build phase capturing the **salient points** from our deep-dive discussions —
questions you raised, design refinements we agreed, tradeoffs, and "aha" points. Updated live as we
talk (loop phase ①).

At the end of a module, these notes are distilled into the **"Salient Points from Our Discussion"**
section of the build summary, so you can recap the conversation, not just the code.

Naming: `phase-<N>-<slug>.md` — e.g. `phase-1-data-storage.md`.
